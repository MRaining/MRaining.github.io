# Grunt配置

在根目录下执行 `npm install` 来安装grunt插件。然后你就可以执行下面的任务来编译less和jade文件。

grunt tasks list

- `grunt`: 编译全部的less和jade文件。
- `grunt watch`: 监控文件改动并自动编译。
